## <a name="overview"></a>概述

此模块在单个位置包含所有不同的 Azure Node.js 模块。

## <a name="install-the-module-with-npm"></a>使用 npm 安装模块

使用 npm 安装用于 Node.js 的 Azure 模块

### <a name="client"></a>客户端

```bash
npm install azure
```
